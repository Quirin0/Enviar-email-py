# Enviar-email-py
**Envie email para qualquer pessoa utilizando python**
<br>

- Como usar:<br>
  1.Coloque seu email em email.txt<br>
  2.Coloque sua senha em senha.txt<br>
  3.Coloque a mensagem que você deseja enviar em mensagem.txt<br>
  4.Coloque os emails que irão receber essa mensagem em lista-emails.txt<br>
  5.python mail.py<br><br>
